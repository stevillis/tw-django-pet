# Sistema de Gerenciamento de Clínica PET

### Funcionalidades

- Cadastro de Clientes
- Agendamento de Consultas para PETs
- Autenticação e Autorização de Usuários

### Tecnologias

- Django
- MySQL (Desenvolvimento)
- PostgreSQL (Produção)
- HTML
- CSS
- Bootstrap
- JavaScript
- jQuery Mask Plugin
- Admin LTE 2
- Git
- Heroku
